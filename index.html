What we do

Testing tools, virtual devices and utilities that make FAT prep, graphics testing and integrations less painful.

Virtual devices

Virtual meters and other simulated devices that behave like real field hardware – so you can build graphics, alarms, logic and trends long before anything ships to site.

Modbus & protocol tools

Modbus TCP servers, register maps and diagnostics for PME, SCADA, BMS and PLC engineers needing predictable, repeatable test sources during commissioning.

Panels & utilities

Compact modular test panels and site-focused utilities to support commissioning teams on critical facilities and integration projects.

Virtual UPS Simulator

A software UPS for EBO and BMS engineers: live voltage, load, battery and runtime values, manual alarms, Modbus TCP and SNMP traps – ideal for testing graphics, alarms and comms before the real UPS is on site.

  • Overview & alarms tabs fully usable for FAT
  • Modbus TCP server with live simulated values
  • SNMP trap sending for key UPS events
  • Perfect for routing, trends & loss-of-comms tests
Request Virtual UPS build
Virtual Energy Meter

A virtual 3-phase energy meter with Modbus TCP and a live dashboard – built for PME, SCADA and PLC engineers who need believable power data on demand.

  • 3-phase V, I, kW, total kW, PF and frequency
  • Simple holding register map (40001–40012)
  • Profiles for office, industrial and random load cases
  • Noise toggle and one-second updates
View Virtual Energy Meter GitHub repo & instructions
