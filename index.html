What we do

Virtual UPS Simulator

A virtual UPS for EBO and BMS engineers: live voltage, load, battery and runtime values, manual alarms, Modbus TCP, and SNMP traps – all without real hardware.

  • Overview & alarms tabs fully usable
  • Modbus TCP server for live UPS values
  • SNMP trap sending for key UPS events
  • Perfect for FAT, graphics, trends & alarm routing
Download Virtual UPS View project on GitHub
Virtual Energy Meter

A virtual 3-phase energy meter with a Modbus TCP server and desktop dashboard. Ideal for SCADA, PLC and PME engineers who need believable test data quickly.

  • 3-phase voltages, currents, kW, PF and frequency
  • Simple holding register map (40001–40012)
  • Profiles for light/office, industrial, and random load
  • Noise toggle and live GUI dashboard
Download Virtual Energy Meter View project on GitHub
