What we do

Virtual UPS Simulator

A software UPS that exposes live voltage, load, battery and runtime, manual alarms, Modbus TCP and SNMP traps – ideal for EBO/FAT testing when the real UPS isn’t on site yet.

  • Overview & alarms tabs fully usable for FAT
  • Modbus TCP server with live simulated values
  • SNMP trap sending for key UPS events
  • Perfect for graphics, alarm routing & trend tests
Request Virtual UPS build
Virtual Energy Meter

A virtual 3-phase energy meter with Modbus TCP and a live dashboard – built for SCADA, PLC and PME engineers who need believable power data on demand.

  • 3-phase V, I, kW, total kW, PF and frequency
  • Simple holding register map (40001–40012)
  • Profiles for office, industrial and random load cases
  • Noise toggle and one-second updates
